# となペンAPI化マニュアル Web記事版

このフォルダは、DOCX校正版から変換したWeb閲覧用ファイルです。

## ファイル

- `docs/index.html`: Web記事本体
- `docs/style.css`: 表示スタイル
- `docs/chapter1.md`: Markdown版
- `source/tonapen_api_manual_chapter1_revised_final.docx`: DOCX原本

## GitHub Pagesで公開する場合

GitHubリポジトリにこの内容を配置し、Settings → Pages で `main` ブランチの `/docs` フォルダを公開元に指定してください。
